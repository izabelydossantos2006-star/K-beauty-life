🎯 Objetivo
Desenvolver uma presença digital voltada para:
Dicas de skincare coreano.
Rotinas de cuidados com a pele.
Tendências de beleza asiática.
Conteúdo educativo e acessível.

🎨 Identidade Visual
A identidade da marca foi construída com base em:
Estilo minimalista.
Paleta em tons suaves (salmão claro e branco).
Tipografia elegante e moderna.
Foco em sofisticação e leveza.

🚀 Tecnologias 
HTML.

💡 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar design, branding e criação de presença digital, unindo estética e estratégia para o nicho de beleza.
